# RayKagi Changelog

## [Initial Version] - {PR_MERGE_DATE}

- **Search Kagi** — free autosuggest as you type (debounced); press `↩` to run a
  Kagi search (results in-app with an API token, browser fallback without).
  `⌘⇧R` runs a deeper search with the `!research` bang.
- **Create Kagi Bang Shortcut** — pick any `!bang` (or a custom trigger) and turn
  it into a Raycast Quicklink you can alias or assign a hotkey.
- **Translate** — one step to your default language; save language-pair presets
  in-extension; bind a pair as a Quicklink for an alias/hotkey.
- **Summarize URL** — open the Kagi Universal Summarizer for a URL.
- **Open Kagi News**, **Random Small Web Page**, **Open Kagi Assistant** — quick
  launchers for Kagi's web tools.
